Animated Robot

an animated robot I built to showcase interactive motion and personality using JavaScript, HTML, and CSS. ByteBot moves, reacts, and comes to life through carefully timed animations and user-driven events, making coding feel playful and expressive.

Pizza Order

Build-a-Pizza — a fun interactive project where users can customize their dream pizza! Using JavaScript, this app lets you choose toppings, sizes, and crusts, then shows your total price in real time. It’s a tasty way to explore user input, DOM manipulation, and basic form logic.

Jungle Adventure

Jungle Quest — an interactive story adventure where you choose your path through the wild! Built with JavaScript, this project lets readers explore dense jungles, face unexpected twists, and make decisions that shape their journey. It’s a creative way to combine storytelling with branching logic and event handling.

Guessing Game

Guess That Number! — a simple but addictive game where players try to guess a random number between 1 and 20. Built with JavaScript, it gives instant feedback, tracks your attempts, and celebrates when you get it right. A great example of using conditionals, loops, and user interaction.

Shopping Game

Grocery Grab — a shopping game where players can add groceries to their cart and see their list build in real time! Made with JavaScript, it lets users select items, view a running total, and practice interactive list management. It’s a playful way to explore arrays, event handling, and dynamic DOM updates.
